# Bouncing Balls

## Development
1. Clone Project to local environment:
   
   git clone git@github.com:danielmihai4/bouncing-balls.git

2. Start dev server:

    npm run dev-server

## Production URL
https://fathomless-savannah-29344.herokuapp.com/

## Deployment
1. Run Command: 

   git push heroku master
